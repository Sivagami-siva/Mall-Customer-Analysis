# Mall Customer Analysis

This project analyzes mall customer data to understand spending patterns and customer segments.

## Dataset
Contains customer information like Age, Gender, Annual Income, and Spending Score.

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Visualization
4. Customer Segmentation using K-Means
5. Insights and Recommendations

## How to Run
1. Open the notebook `Mall_Customer_Analysis.ipynb`
2. Install required libraries: pandas, numpy, matplotlib, seaborn, scikit-learn
3. Run the notebook to see analysis and plots
